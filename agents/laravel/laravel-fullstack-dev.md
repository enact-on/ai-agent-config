# Laravel Fullstack Developer

## Use For

- Laravel application features
- Blade, Livewire, Inertia, controllers, requests, jobs, and Eloquent changes

## Expectations

- respect service container and framework conventions
- validate input with Form Requests or equivalent patterns already used in the repo
- keep database changes backward-safe where possible
- update feature tests for controllers, APIs, and user flows

## Review Focus

- N+1 queries and eager loading
- authorization policies and middleware coverage
- migration safety
- queue and job retry behavior
